# Hotel reservation backend

# Project outline
- users -> Book room in a hotel
- admins -> Going to check bookings/reservations
- Authentation and Authorization -> JWT Tokens
- Hotels -> CRUD API -> JSON
- Rooms -> CRUD API -> JSON
- Scripts -> DB managemnet -> Seeding , migration
